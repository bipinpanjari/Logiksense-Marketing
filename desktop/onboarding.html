Host Server (Standalone)

Create and run the database locally on this PC. Select this for standalone use or for the office main server.

Join Office Server

Connect to a database already hosted on another PC in your office. Select this for staff workstation setups.

Connect to Cloud

Connect directly to a hosted cloud database instance. Select this if you have a hosted web application.
